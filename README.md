# Игра Камень-Ножницы-Бумага

## Описание
Простая консольная реализация классической игры "Камень-Ножницы-Бумага" (Rock-Paper-Scissors) на Python. Игра включает в себя подсчет очков с сохранением результатов между сессиями.

## Как играть
1. **Запуск:** Откройте терминал в директории проекта и запустите игру:
   ```bash
   python game.py
   ```
2. **Ход:** Введите соответствующую букву для своего хода:
   * `r` (Rock/Камень)
   * `p` (Paper/Бумага)
   * `s` (Scissors/Ножницы)
3. **Выход:** Введите `q` для выхода из игры.

## Функциональность
* Случайный выбор хода компьютера.
* Обработка ввода пользователя (проверка на корректность).
* Определение победителя.
* **Система подсчета очков:**
    * Сохранение счета побед/поражений/ничьих в файле `scoreboard.txt`.
    * Отображение текущего счета.

## Требования
* **Python 3.x**
